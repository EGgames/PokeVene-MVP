# PokeVene MVP

Juego web de trivia donde el usuario debe identificar si un termino es un **Pokemon** o un **termino venezolano**.

## Stack Tecnologico

| Capa | Tecnologia |
|------|-----------|
| Frontend | React 19 + Vite |
| Backend | Node.js (Express) + JavaScript |
| Base de datos | PostgreSQL |
| Autenticacion | JWT (backend-only, sin Firebase) |
| Testing | Jest (backend) - Vitest (frontend) - Serenity BDD + WebDriver (E2E) |
| CI/CD | GitHub Actions |

cd backend && npm install && npm run dev
